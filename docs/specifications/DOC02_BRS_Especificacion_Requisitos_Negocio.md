# Especificación de Requisitos de Negocio (BRS) - SocialHub

## 1. Introducción
### 1.1 Propósito del negocio
Ofrecer una alternativa segura y minimalista a las redes sociales actuales, enfocada en la interacción directa sin algoritmos de manipulación.

### 1.2 Alcance del negocio
Usuarios finales y organizaciones que requieren una plataforma de microblogging privada.

### 1.3 Stakeholders principales
- Usuarios de la comunidad CONFIANZA23.
- Administradores de TI.

## 2. Referencias
- CONF23-STD-SDLC-006.
- OWASP Top 10 2025.

## 3. Requisitos de Gestión de Negocio
### 3.1 Misión y Objetivos
- Misión: Democratizar el microblogging seguro.
- Objetivo: Alcanzar 1000 usuarios activos en el primer trimestre.

## 4. Requisitos Operativos de Negocio
### 4.1 Procesos de Negocio
- Publicación de contenido.
- Interacción social (Likes/Comentarios).
- Gestión de perfil.

## 5. Concepto Operativo Preliminar
El sistema operará como una aplicación web centralizada con capacidad de visualización móvil escalable.
