# Definition of Done (DoD) - SocialHub

## Estándares de Codificación
- Cumplimiento estricto de **PEP 8**.
- Tipado estático con **PEP 484/526**.
- Docstrings en formato Google.

## Requisitos de Pruebas
- Pruebas unitarias con cobertura > 80%.
- Verificación de seguridad con `ast`.
- Validación de indentación con `tabnanny`.

## Proceso de Revisión
- Revisión de código side-by-side.
- Aprobación del Product Owner.

## Documentación
- Actualización de ADR si hay cambios arquitectónicos.
- Documentación técnica generada.
